# Proposal PROP-20260309-TOOLS-001

- Date: 2026-03-09
- Candidate Target: TOOLS.md
- Type: tools-rule
- Risk: low
- Reviewer Decision: pending

## Summary
Use spreadsheet-native tooling for spreadsheet tasks before document tools.

## Why
Several spreadsheet tasks were slower or less accurate when routed through non-spreadsheet workflows.

## Evidence
- RTE-20260302-001
- LRN-20260304-004
- ERR-20260306-002

## Suggested Text
For spreadsheet work, prefer spreadsheet-native tooling first. Avoid document or PDF workflows unless explicitly requested.

## Rollback
Remove if tooling architecture changes and the rule becomes obsolete.
