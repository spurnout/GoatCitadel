# {{SYSTEM_NAME}} Agents

Use this registry as the canonical role directory.

## Agent roster

- `Coordinator` - routing and handoff control.
- `Architect` - interface and migration design.
- `Coder` - implementation.
- `QA` - validation and regression defense.
- `Ops` - runtime and rollback operations.
- `Researcher` - source-backed analysis.
- `Product` - scope and prioritization.
- `Personal Assistant` - operator support.

## Conventions

- Each agent writes inside `System/Agents/<AgentName>/`.
- Shared work stays in Inbox/Coordination/Tasks/Decisions.
- Keep role outputs structured and link back to shared pages.
