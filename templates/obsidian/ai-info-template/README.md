# {{SYSTEM_NAME}} Obsidian Template

This is an optional vault starter for users who want to run {{SYSTEM_NAME}} with Obsidian.

## What you get

- Shared operating pages: `Inbox`, `Coordination`, `Tasks`, `Decisions`.
- Agent registry + per-agent folders.
- Daily operating rhythm and handoff conventions.

## Placeholders

- `{{SYSTEM_NAME}}` - Name of the AI system.
- `{{WORKSPACE_NAME}}` - Active workspace label.
- `{{DATE}}` - Scaffold date.

## Notes

- This template is optional.
- GoatCitadel does not require Obsidian.
- Keep vault write access restricted to your chosen subpaths.
